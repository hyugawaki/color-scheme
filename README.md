# Color Scheme Generator

カラーパレットを作成する配色ジェネレーターです。

## デモ

![Color Scheme Generator Screenshot](./src/assets/png/website.png)
[Color Scheme Generator](https://hyuga-colorschemegenerator.netlify.app/)

## 使い方

1. 好きな色を1色選びます
2. テーマを選択します
3. "Get color scheme"を押します
4. 5色、配色されます
